dotNetRDF BSBM Server

This web application provides endpoints for running the Berlin SPARQL Benchmark against at a variety of scale factors:

  • 10
  • 50
  • 100
  • 250
  • 500
  • 1000
  • 1500
  • 2000
  • 2500
  • 2785

You may add the word stats on to the end of any of the above URLs to use the Stats based optimiser instead of the default optimiser.
